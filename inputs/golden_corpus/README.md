# Golden Corpus (Parsing Benchmark)

This folder contains a small, **synthetic** PDF corpus used to benchmark parsing quality and speed.
All files are generated locally (no copyrighted content) and are intentionally "tricky".

## Files
- `scan_like_blank_3p.pdf`: Blank pages (simulates scanned/OCR-required PDF). Expect empty text.
- `two_column_mock.pdf`: Two-column layout with separate text blocks.
- `math_heavy_mock.pdf`: Math-heavy content (ASCII formulas).
- `thesis_long_10p.pdf`: Long document with 10 pages.
- `tables_mock.pdf`: Table-like alignment with pipes/spaces.
- `hyphenation_wrap.pdf`: Hyphenated words split across lines.
- `references_dense.pdf`: Dense reference list with DOIs.
- `figure_captions.pdf`: Figure caption patterns.
- `headers_footers.pdf`: Repeated headers/footers across pages.
- `mixed_layout.pdf`: Mixed bullets + short paragraphs.
